Hello,

   To use your own Datafile create a new directory e.g. worldcup

   Next create the Datafile in the worldcup direcotry - it's just a
regular text file with the name Datafile

  Add:

world  'world.db', setup: 'countries'
football 'national-teams'
football 'national-teams', setup: 'history'
football 'world-cup', setup: 'history'

   to the Datafile.

   And finally build your own database use the sportdb machinery. Type:

   $ sportdb build

   And that's it. The build command will look for a Datafile in the
working / local directory and start downloading the datasets / zips /
archives and build the sqlite database (called sport.db by default).

   Enjoy the beautiful game. Cheers. Prost.
