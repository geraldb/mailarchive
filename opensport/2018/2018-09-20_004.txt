Hello,

    Thanks for your kind word. Welcome to football.db.

     Sorry the /br-brazil repo [1] or results are currently without
maintainer / updates. Thanks to Rafael Pereira [2] for adding most of
the match schedules and results (in the past).

    The repo is setup to be "self-contained", that is, all you need is
the clone the single country repo and than use the Datafile [3] to
build a copy e.g.

   $ cd br-brazil
   $ cat Datafile        # check Datafile, that is, build script
   $ sportdb build     #  will use (default) to the "local" Datafile

  That's it. The readme also has instructions :-). Cheers. Prost.

[1] https://github.com/openfootball/br-brazil
[2] https://github.com/rlpereira
[3] https://github.com/openfootball/br-brazil/blob/master/Datafile
