  Sorry there's no definite step-by-step guide. It's mostly learning
by doing and by looking at "real-world" examples. The first step is to
build yourself an SQL database from the datasets (text files) with the
build script / command line tools (e.g. sportdb).

more leagues, more teams, etc.) and / or double checking to see if all
results and team are in order and so on.
