I tried working on this a little.  I found the following satisfied the 
build tasks if you are using openfootball/build[1].

Add the following to build/tasks/setups/cl.rake:

task :cl201415  => :importbuiltin do
  SportDb.read_setup( 'setups/clubs',    CLUBS_INCLUDE_PATH )
  SportDb.read_setup( 'setups/clubs',    AT_INCLUDE_PATH )
  SportDb.read_setup( 'setups/clubs',    DE_INCLUDE_PATH )
  SportDb.read_setup( 'setups/clubs',    EN_INCLUDE_PATH )
  SportDb.read_setup( 'setups/teams',    ES_INCLUDE_PATH )
  SportDb.read_setup( 'setups/teams',    IT_INCLUDE_PATH )
  SportDb.read_setup( 'setups/2014_15', 
 EUROPE_CHAMPIONS_LEAGUE_INCLUDE_PATH )
end

Comment out the following teams (or alternatively properly add them in the 
respecitve repos) in 2014-2015/cl.yml and cl_quali.yml    

    paris

    monaco

    aktobe

    lille

    salzburg

    grasshopper

    athletic

    leverkusen

    basel
Couldn't push those to the build and champions league repos respectively 
bcs I don't have permission :)

Then, you can use the build repo[1] to build:

rake build DATA=cl201415

Hope that gets you going!  Looks like there's probably more cleanup to be 
done.
