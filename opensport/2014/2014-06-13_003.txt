Hello,

> Is it possible for me, to update the score in your database?
> If it's possible, can you explain me how can I have to do?

  It works like a wiki (sort of). You update (e.g. add the results) to
the plain text files in the repos.

  For example,  world-cup/2014--brazil/cup.txt  - before:

(1) Thu Jun/12 17:00   Brazil - Croatia

  After:

(1) Thu Jun/12 17:00   Brazil 3-1 Croatia

  That's it.

   If you're member you can edit/update the files without pull request
(in your browser via GitHub or your local copy and then push changes
upstream etc.).   If you're not yet a member you have to send a pull
request.  Cheers.
