Hello,

  If you're wondering - What's RSSSF? What's the Rec.Sport.Soccer
Statistics Foundation?

  The RSSSF collects and offers football (soccer) league tables from
all over the world in plain text.

Example:

   Round 1
   [May 25]
   Vasco da Gama   1-0 Portuguesa
    [Carlos Ten=C3=B3rio 47']
   Vit=C3=B3ria         2-2 Internacional
    [Maxi Biancucchi 2', Gabriel Paulista 11'; Diego Forl=C3=A1n 29', Fred =
63']
   Corinthians     1-1 Botafogo
    [Paulinho 73'; Rafael Marques 24']
   ...

  More info at the project site @ www.rsssf.com

  Now of course the question is - if it's all plain text - why not
read the match fixtures and add the results to the football.db (that
is, to SQL tables)?

  If you're interested I started (yet another) GitHub org that hosts
RSSSF archive data that you can read into your SQL database of choice
(SQLite, PostgreSQL, etc.). Still early and a little rough - as always
expect surprises.

   More info @ github.com/rsssf

  Cheers.
