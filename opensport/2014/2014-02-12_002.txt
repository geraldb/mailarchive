Hello,

  FYI: I've added a new grounds table (for stadiums/venues). Plus the
games table and the events table now include references for grounds.

  For example, using the World Cup event in Brazil 2014 you can get
all twelve stadiums by using:

   event =3D Event.find_by key: 'world.2014'
   event.ground.count  # =3D> 12
   event.grounds       # =3D> [
     'Est=E1dio do Maracan=E3 -- Rio de Janeiro',
     'Est=E1dio Nacional Man=E9 Garrincha -- Bras=EDlia',
     'Arena Corinthians -- S=E3o Paulo',
     ..
   ]

   To get the city for a game, for example, use:

   # (1) Thu Jun/12   Brazil - Croatia       @ Arena de S=E3o Paulo, S=E3o =
Paulo

   game.ground.title        # =3D> Arena de S=E3o Paulo
   game.ground.city.title   # =3D> S=E3o Paulo

   And so on. See the World Cup 2014 fixtures [1], for an example for
plain text games/match fixtures with stadium/venue location included.

   Cheers.

[1] github.com/openfootball/world-cup/blob/master/2014/cup.txt
