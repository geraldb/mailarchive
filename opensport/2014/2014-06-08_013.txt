Hello,

   Thanks for your interest in football.db.  You can get pre-built 
databases on the build release page [1] - for the World Cup in Brazil -  
download the worldcup2014.db file.

    Of course - with more experience (later) it's best to build your own 
copy (to make sure you get up-to-date data and include what you need etc.)

    You can start as an alternative the service using the sportdb command 
line tool e.g.

$ sportdb serve    # assume default database name (sport.db)  rename or 
change via -d switch e.g.
$ sportdb -d worldcup2014.db serve

    Good luck. Cheers.

[1] github.com/openfootball/build/releases
