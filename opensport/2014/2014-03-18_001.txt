Hello Christos,

   The pre-built SQLite Database for Download already has the data
sets (files) imported into an SQL Database, that is, football.db.
Use the sqlite3 command to query the db. Check if the sqlite3 tool is
installed e.g.

  $ sqlite3 -help

  And to query use, for example:

  $ sqlite3 football.db
  sqlite> select * from teams;

  and so and so forth.
  Learn more about SQLite @ www.sqlite.org/sqlite.html or try visual
tools like SQLite Studio (free) and friends.  Cheers.
