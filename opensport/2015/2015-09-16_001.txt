Hello,

  Archit Verma has put together a command line tool (in Python) called
soccer-cli [1] that lets you query an online HTTP JSON API for league
standing tables, match scores and more.

  For example, to show the latest English Premier League matchday use:

    $ soccer --league=EPL

  To get the standings table use:

    $ soccer --standings --league=EPL

  To get scores for your favorite club (e.g. Manchaster Utd) use:

    $ soccer --team=MUFC

  And so on.

  If you try out the soccer-cli tool, let us know what you think. Know
any other football (soccer) command line tools, any tips welcome.

  Cheers.

PS: I try to collect all tools on the awesome-football page [2].
