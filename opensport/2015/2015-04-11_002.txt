Hello,
   Thanks for trying the build.sh script.  And thanks for the details
error reports.

  For debugging try the --verbose option/switch. For example:

> $ sportdb --dbname football.db update --include 'stadiums'

change to

$ sportdb --verbose --dbname football.db update --include 'stadiums'

  That should result in a more detailed error report (you're invited
to post the new dump.)

  In general it's recommended to use the new build system, that is,
the easiest and recommended (yes, best) way is using datafiles. See
github.com/openfootball/datafile

  Cheers.
