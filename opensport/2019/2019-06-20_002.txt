  I've added a new (quick) importer for datafiles and packages in the

  For example, to import all datafiles for all seasons for England [1], use:


  require 'sportdb/import'

  SportDb.connect( adapter:  'sqlite3',
                   database: './eng.db' )

  ## build schema
  SportDb.create_all

  ## turn on logging to console
  ActiveRecord::Base.logger = Logger.new(STDOUT)

  pack = CsvMatchImporter.new( '../england' )
  pack.import_leagues
