Hello,

> How can I specify that I want the rows inserted in a postgresql database =
and not sqlite?

  Welcome to the football.db.  Search the forum for postgresql :-).
See for example "Help for a beginner by Guillaume Bleuz=C3=A9".

  To quote:
   You can read the datasets into your (SQL) database of choice e.g.
postgresql. However, you, will have to learn to use a (build) script
on the command line (in ruby). Example:

 build.rb:

require 'sportdb/models'
require 'datafile'


DB_CONFIG =3D {
  adapter:  'postgresql',
  encoding: 'unicode',
  username: 'you',
  password:  'topsecrect',
  pool:      5,
  timeout:   5000
}


ActiveRecord::Base.establish_connection( DB_CONFIG )


SportDb.create
SportDb.read_builtin


datafile =3D Datafile::Datafile.load_file( "./worldcup.rb" )

datafile.dump

datafile.download    ## download zips from github - saved to ./tmp
datafile.read            ## read into / import into sql database


   If you run this build script with ruby and you install the ruby
libraries / gems (sportdb-models, datafile) and if you use the
worldcup datafile, you're all set.

  $ ruby ./build.rb

   Good luck. Cheers. Prost.
