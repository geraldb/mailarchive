>  how can I contribute in filling data: do I simply need to update the text files > in https://github.com/openfootball/euro-cup?

   Yes, the idea is that it works like a "modern" wiki e.g. a text
file anybody can edit.

   The difference from a "standard" wiki is using git and github e.g.
the first time if you're not a member you have to send in a "patch",
that is, pull request.

   Using the tooling e.g. the spordb command line tools you can turn
the "plain text files" into a proper SQL database (e.g. football.db).
