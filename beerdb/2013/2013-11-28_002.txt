   Sorry this might be confusing.  The activerecord-sqlite3-adapter is 
bundled with (already included) in the standard activerecord gem. There is 
no activerecord-sqlite3-adapter gem. Thus, try to install activerecord e.g.:

   $ gem install activerecord    

   and also make sure you have the sqlite3 gem installed e.g.:

   $ gem install sqlite3

    Now try again. Let me know if it works. Cheers.   


PS: The activerecord-jdbc-sqlite will NOT work because it requires the Java 
version of Ruby (that is, JRuby). The exception, of course, is if you 
actually run on Java.  (jdbc == Java DB Connectivity)
