Hello,

  FYI: I've put together a new world data gem, that is, worldlite,
that includes all the world countries (and territories) as good ol'
ruby code. Example:

module WorldLite

  c =3D Country.new
  c.name   =3D 'Austria'
  c.key    =3D 'at'
  c.alpah2 =3D 'AT'     # iso two-letter country code (ISO 3166-1 alpha-2)
  c.alpha3 =3D 'AUT'    # iso three-letter country code (ISO 3166-1 alpha-3=
)
  c.fifa   =3D 'AUT'    # football country code (fifa =3D F=C3=A9d=C3=A9rat=
ion
Internationale de Football Association)
  c.ioc    =3D 'AUT'    # olympics country code (ioc =3D International
Olympic Committee)
  c.net    =3D 'at'     # internet top level domain
  c.motor  =3D 'A'      # motor vehicle license plate code
  c.num3   =3D '040'    # iso numeric three-digits code as string (ISO
3166-1 numeric)
  c.num    =3D 40       # iso numeric code as number

  c.continent_name =3D  'Europe'

  c.un     =3D true     # United Nations member?  -- 193 member countries
  c.eu     =3D true     # European Union member?  -- 27 member countries
  c.euro   =3D true     # Euro Zone member?       -- 17 member countries

  c.wikipedia =3D 'Austria'   # e.g. see en.wikipedia.org/wiki/Austria
  c.wikidata  =3D 40          # e.g. see wikidata.org/wiki/Q40
  c.factbook  =3D 'au'        # e.g. see
www.cia.gov/library/publications/the-world-factbook/geos/au.html

end  # module WorldLite

  The worldlite gem is lightweight, that is, no database and no
dependencies (extra gems) required. All code and data is public
domain, that is, license-free - no copyright and no rights reserverd -
use as you please.

   To try it - type in your Ruby shell:

   require 'worldlite'
   include WorldLite

   WORLD.size         # =3D> 245
   WORLD_UN.size   # =3D> 193
   WORLD_G20.size  # =3D> 20

   and so on. For countries try:

   AT.class.name  # =3D> WorldLite::Country
   AT.name           # =3D> Austria
   AT.alpha3         # =3D> AUT

   Still early and rough but all the world's countries and territories
(245 total) included. Cheers.
