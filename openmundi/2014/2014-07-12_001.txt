Hello,

  FYI: I've put together a factbook gem [1] that lets you read in
World Factbook country profile pages (in HTML) and returns a
structured Ruby hash that you can save, for example, as a pretty
printed JSON file. Example:

 Get country profile page for Austria as a hash (that is, structured
data e.g. nested key/values)

   page = Factbook::Page.new( 'au' )
   pp page.data          # pretty print hash

  Save to disk as JSON

   File.open( 'at.json', 'w') do |f|
     f.write page.to_json( pretty: true )
   end

  and so on.

  Find ready-to-use country profiles in JSON in the
openmundi/factbook.json [2] repo.

  Cheers.

[1] github.com/worlddb/factbook.ruby
[2] github.com/openmundi/factbook.json
