Hello,

  Thanks for your kind words and your interest in factbook.json. 

1) About the update frequency: The HTML scraper script is open and public 
domain (see the factbook gem) [1] To (re)generate the JSON documents just 
takes a couple of minutes. If you need an update it's best to generate your 
own up-date-date dataset (and ideally push back your changes to update it 
for everyone). It's an open source all-volunteer efforts open to anyone. 

2) About more countries: For now only the 193 UN countries are included 
(plus the world and the European Union). To add more countries / 
dependencies use the script. For example, to add Hong Kong use:

    page = Factbook::Page.new( 'hk' ) # hk is the country code forHongKong
    pp page.data # pretty print hash

And so on. Cheers.

[1]  github.com/worlddb/factbook.ruby
