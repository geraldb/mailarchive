Hello,

  FYI: If you use the "old" build script using Rake e.g. Rakefile [1]
you can import the data into MySQL. Change the db settings e.g.:

DB_CONFIG = {
  adapter: 'sqlite3',
  database: WORLD_DB_PATH
}

  to something like:

DB_CONFIG = {
  adapter: 'mysql',
  host: 'localhost',
  database: 'world',
  username: 'root',
  password: 'secret'
}

  Cheers.

[1] https://github.com/openmundi/build/blob/master/Rakefile
