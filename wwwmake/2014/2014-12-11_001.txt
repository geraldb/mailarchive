Hello,

  I've updated the slideshow-models machinery, that is, v2.5.0. What's news?

  Now you can use npm [1] to install slideshow template packs. For
example to install the slideshow-s6-blank pack type:

  $ npm install slideshow-s6-blank

   No magic here. The npm package manager will download and copy the
package to your working folder in the node_modules/ folder.

   That's it. Now the template pack is ready to use with Slide Show
(S9). To double check type:

  $ slideshow ls

   Will print something like:

Installed template packs in search path
    [1] templates/*.txt
    [2] templates/*/*.txt
    [3] node_modules/*/*.txt
  include:
         s6blank (node_modules/slideshow-s6-blank/s6blank.txt)
         ...

  To use the new template pack add as usual the -t/--template option e.g.

  $ slideshow build welcome.text --template s6blank

  Note: I've published most template packs on npm. Search for
slideshow-templates, s9 or similar [2].

  Cheers.

PS: To update the slideshow-models gem type: $ gem update slideshow-models

[1] docs.npmjs.com
[2] www.npmjs.com/browse/keyword/slideshow-templates
