I've put together an s6 template that aims to be clean and simple. It
is black and white---no fancy gradients, no colorful headers or
footers. It tries to improve legibility by avoiding long horizontal
runs of text. It also styles some of the features provided by pandoc---
e.g., definition lists.

You can find it here:

http://github.com/dsanson/s6clean

Though I haven't tried it, I assume you can install it with

slideshow -f http://github.com/dsanson/s6clean/raw/master/s6clean.txt
