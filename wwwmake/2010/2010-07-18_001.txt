   I've updated the S6 templates and scripts. What's new?

   == JavaScript Code-Cleanup Including New Options: Autoplay, Title
Selection, Transition

    All the S6 machinery for the slide show now resides in a single
JavaScript file, that is, jquery.slideshow.js.

    If you want to customize the code use the new options or add your
own JavaScript code inside a document.ready block.

    The standard "plain" setup is now:

    $(document).ready( function() {

      Slideshow.init();
    }

    If you want to use, for example, a custom transition (e.g. using a
scroll up effect), set the transition property to point to your own
function (using fn($from,$to)). Example:

    Slideshow.transition = transitionScrollUp;

    Slideshow.init();

    To start a slide show in autoplay mode, use the mode option:

    Slideshow.init( { mode: 'autoplay' } );

    See an example with autoplay and scroll up transition online @
http://slideshow.rubyforge.org/tutorial.autoplay.html

    The autoplay machinery is now built-in. Use the a/s/p-keys to
toggle (start/stop/pause) autoplay.

    To change the title selector from h1 to h2 use:

    Slideshow.init( { titleSelector: 'h2' } );


   == Heading Clicker Added

    You can now also go to the next slide by clicking on the heading.

   == Internet Explorer Better Browser Banner

   I've moved the better browser banner for IE out of the core and the
banner is now just a single stand-alone JavaScript module (that is,
jquery.microsoft.js)

   If you want to exclude it from your templates, remove the
conditional comment, that is:

<!--[if IE]>

<script src="shared/jquery.microsoft.js"></script>

<![endif]-->

   That's it. You can find the S6 Blank package  @ http://github.com/geraldb/s6

   Cheers.
