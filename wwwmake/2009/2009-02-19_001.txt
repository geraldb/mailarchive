Hello,

  I added the missing support for incremental display (also known as
steps) to S6 [1]. I also updated the S9 gem [2] to include the updated
S6 templates.

  What's new?

  Use the step class to mark items you want to display step-by-step.
Note, you can mark lists (e.g. ul,ol) to mark all items at once e.g.

   <!-- mark list with class step to mark all items at once -->
    <ul class='step'>
      <li>Item 1.1 Here</li>
      <li>Item 1.2 Here</li>
    </ul>

    <!-- or mark individual list items -->
    <ul>
      <li class='step'>Item 2.1 Here</li>
      <li class='step'>Item 2.2 Here</li>
    </ol>

  To style your steps use the step class for upcoming steps (not yet
displayed) and stepcurrent for the current step. Example:

  .step { color: gray; }  /* or hide next steps e.g.  .step {
visibility: hidden; } */

  .stepcurrent { color: yellow; }


  If you use S9 to generate slide shows w/ Textile mark items with
(step) the Textile syntax for adding classes to tags. Here's an
example:

h1. Getting Started in 1-2-3 Easy Steps

*(step) Step 1: Author your slides in plain text using a wiki-style
markup language
* Step 2: Generate your slide show using the @slideshow@ gem
* Step 3: Open up your slide show in your browser and hit the space
bar to flip through your slides
* That's it. Showtime!

h1. Gradient Themes

p(step). Q: How To Use Built-In Gradient Themes?

p(step). A: You can define your gradient theme in plain text in the
slide show source in the header using a simple CSS-style name value
pair.

   That's it. Questions and comments welcome. Cheers.
