Hello,

   I've added some more updates to the S6 Blank Templates[1]. What's
new?

   * Added a fix/workaround for Chrome for inline styles (now using
media='screen,projection' instead of media 'projection'; projection
only won't work in chrome for style switching).

   * Moved code to addons using new slideshow custom events (triggers)
such as slideshow.change,
     slideshow.init, slideshow.start, slideshow.keys, etc.; New addons
include:

     * counter (slide counter e.g. 1/7)
     * controls (toggle, next, prev buttons, jumplist)
     * autoplay
     * footer/header

  * Moved sample transitions to jquery.slideshow.transitions.js
  * Changed box-sizing for .slide to border-box that lets you use
simply use width: 100%; height: 100%;
  * Some minor fixes.

  That's it. Questions and comments welcome. Cheers.
