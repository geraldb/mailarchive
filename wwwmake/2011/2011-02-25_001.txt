   That's definitely the recommend way because it's just "plain" CSS3.
The only downside is that it only works with modern browsers and you
might have to change your code as the CSS3 syntax changes and vendors
drop the -moz and -webkit prefix.

   As an exercise you might write your own gradient helper to generate
the CSS3 code. As an example see
https://github.com/geraldb/slideshow/blob/master/lib/slideshow/helpers/back=
ground_helper.rb

   As an alternative you might use jQuery to add the css class to all
slides e.g.

    $( '.slides' ).addClass( '.gradient_yellow_orange' );
