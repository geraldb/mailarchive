Hello,

1. Where do I get 'slideshow gem?

  Use the package manager tool bundled with Ruby e.g. issue

  $ gem install slideshow

  Try first if the gem tool is available, for example, issue:

   $ gem env       or
   $ gem list slideshow -r

   That's it.  After the install try

   $ slideshow help

2. Is it compatible with a Drupal website?

    The generated slideshow is static, that is, it's just a bunch of files, 
that is, HTML, CSS, JS, plus images. No content management system or 
application server or scripting runtime (other than JavaScript, of course) 
needed for hosting on the server. Please, check the Drupal docu how to 
handle static web content, I don't know what's the best practice. 

    All the best. Good luck. Cheers.
