Hello Lee S,

  "Out-of-the-box" or "out-of-the-gem" has no built-in support for
math equation. You have a couple of options:

  o Add a JavaScript math equation library/package to your template to
add math equation in plain text or tex. Examples include:


  -   ASCIIMathML =96 [1] =96 Converts expressions/formulas in ASCII
(plain text) to MathML
  -   jsMath =96 [2] =96 Converts TeX expressions/formulas to HTML
  -  MathJax =96 [3] =96 Converts Tex or MathML expressions/formulas to HTM=
L/CSS

   I'm not an expert so I can't recommend a package. Also I'm sure
there are more options.

  o Add a Ruby plugin to convert math equation to MathML or similar.

  o Use pandoc as your markdown processor engine with slideshow (NB: I
haven't tried the pandoc plugin for a while - might need an update -
you find the code here included via the markdown gem [4]

    That's all I can think of now. Let us know what works for you. Cheers.

[1] http://asciimathml.sourceforge.net
[2] http://www.math.union.edu/~dpvc/jsMath
[3] http://www.mathjax.org

[4] https://github.com/geraldb/markdown/blob/master/lib/markdown/engines/pa=
ndoc_ruby.rb
