Hello,

  Fantastic. Thanks for sharing. I've added the template pack to the 
gallery [1] and I've also added a shortcut to the update index [2].

   Use 

  $ slideshow update    # to update the shortcut index

   than you can use

  $ slideshow install reveal

   as a shortcut instead of

  $ slideshow install 
https://raw.github.com/avillafiorita/slideshow-reveal.js/master/reveal.txt


   Cheers.


PS: You might consider changing the manifest from
  reveal.txt to reveal.js.txt   The manifest name becomes the name of the 
template pack e.g

It lets you use

slideshow build welcome.text -t reveal.js

instead

slideshow build welcome.text -t reveal


[1] http://slideshow-s9.github.io/gallery.html#revealjs
[2] https://github.com/slideshow-s9/update
