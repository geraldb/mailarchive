  What's the markdown gem?

   The Markdown Engine Wrapper (markdown) Ruby gem used by the Slide
Show (S9) gem and others lets you use your markdown library of choice
using ENV or markdown.yml settings.

  What's new?

markdown web service/HTTP API running on Heroku @ http://hypertext.herokuapp.com

  or start your own server issue the command:

   markdown server  or
   markdown s
