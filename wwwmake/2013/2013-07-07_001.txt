Hello,

  I've updated the Slide Show (S9) gem. What's new?

  I've added support for light-weight plugins that you can install the same 
way as template packs. Example:

     slideshow install analytics   

     or
  
     slideshow install 
https://raw.github.com/slideshow-s9/analytics/master/analytics.txt.plugin

   This will install the analytics helper plugin [1]. Now you can use it in 
your slides:

     {{ google_analytics code=UA-YOUR-CODE-HERE }}

    To create your own plugins add a manifest to your repo (e.g. 
analytics.txt.plugin) that lists your Ruby scripts (e.g. 
analytics_helper.rb).

   That's it. Enjoy. Cheers.

[1] https://github.com/slideshow-s9/analytics
