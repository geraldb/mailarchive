Hello,

  I've uploaded a new Slideshow (S9) gem, that is, v0.9.11. What's
new?

  Gradients via headers are back in the builtin S6 template pack. I
added a new helper, that is, gradient_from_headers[1], that will
generate and add the proper CSS3 style rule if you set any of the
gradient headers, that is, gradient, gradient-theme or gradient-
colors.

  For example, try:

   gradient: top-bottom red black

   and it will now work again as advertised on the gradient themes
page[2] if your browser supports CSS3 gradients. See the "When can I
use" site[3] or try the gradient sample page (using CSS3)[4] to check
up on browser support (e.g. Internet Explorer 10+, Firefox 3.6+, Opera
11.1+, Chrome 10+ etc.).

   Cheers.
