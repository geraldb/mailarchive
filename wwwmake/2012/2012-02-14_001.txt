Hello Mark,

    Sorry for the confusion. As posted in an older thread:

  The old way of gradient support using SVG is discouraged (outdated
to be honest and, thus, removed from the S6 Blank template pack). It's
far easier to use "plain" CSS3 gradients. There's nothing special
about it just add the CSS3 gradients to your stylesheet.

    Example:

%css
.slide {
    background-image: -webkit-gradient(linear, 0% 0%, 0% 100%,
from(blue), to(black));
    background-image: -moz-linear-gradient(top, blue, black);
}
%end

  Let me know if that makes sense and works for you. Cheers.

PS: Tip: You can use any of the free online CSS3 gradient builde.
Ironically, Microsoft has a good one at ->
http://ie.microsoft.com/testdrive/Graphics/CSSGradientBackgroundMaker/
, for example.
