  Not sure excatly what you mean.

  The step helper is in Ruby code[1] and runs on the server and justs
adds a div block with the  step css class (e.g. <div class='step'>) to
your html. You can create your own Ruby helpers either inline using
the %helper %end block or saving your Ruby helper code in the /lib
folder.

   I do not know deck.js in detail but I assume it already supports
steps. Please, check the deck.js docu to find out what css class it
may use.
