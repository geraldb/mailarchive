Hallo,

  I've updated the Slide Show (S9) gem and added a new switch that
makes it easier to use the Takahashi method [1] for your slides.

  To break up words into slides the new --takahashi switch lets you
use // between words or on its own line. Example:

%%%%%%%%%%%%%
%% here we go

Welcome // Aloha // Bienvenue // Benvenuto
//
V=C3=ADt=C3=A1me t=C4=95
//
Shalom
//
Ahlan'wa sahla
//
Bem-vindo // Bem-vinda // Bienvenido
//
Willkommen // Welkom // Velkommen // Wolkom
//
=CE=9A=CE=B1=CE=BB=CF=8E=CF=82 =CE=BF=CF=81=CE=AF=CF=83=CE=B1=CF=84=CE=B5
//
Dobrodo=C5=A1li

   If you add some styling (to make your text bigger or better huge)
use a %css directive. Example:

%%%%%%%%%%%%
%% some extra styling

%css

p {
   margin: 0;
   padding: 12% 0 0 0;
  text-align: center;
  font-size: 160px;
}

%end


  Showtime! Issue the command:

   slideshow --takahashi taka.text

  That's all. If you have any question or comments, let me know.
Cheers.

[1] http://en.wikipedia.org/wiki/Takahashi_method
