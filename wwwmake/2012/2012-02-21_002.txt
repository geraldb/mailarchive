  Tip: Check the generated html file. Does it have proper img tags?
e.g.

  <img src="c:/IMG_0679.jpg">

  If yes, than I guess the browser cannot handle the Windows file path
(that is, c:/). Change c:/ to file:///c:/

  Example:

  <img src="file:///c:/IMG_0679.jpg">
  ![Kids](file:///C:/IMG_0679.jpg)
