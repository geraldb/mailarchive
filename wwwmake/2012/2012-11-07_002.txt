> How do you set this css parameter?

  It works like in Ruby on Rails views/layouts if that helps.
Basically, to add content use it as a block (in a different template
e.g. slides.html.erb). Example:

   <% content_for :css %>
     your css here
   <% end %>

   That's really all there is to it. More info [1].

   Cheers.

[1] http://guides.rubyonrails.org/layouts_and_rendering.html#using-the-content_for-method
