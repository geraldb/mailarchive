> =A0 =A0The installation troubles are caused by the Oniguruma gem [1] (a
> new regular expression engine) that is required by the Ultraviolet gem
> [2] (a syntax highlighting engine).
>
> =A0 On Windows the Oniguruma gem includes the pre-built C extension
> required for the gem. However, on all other systems - including OS X -
> the Oniguruma gem assumes you have the Oniguruma regular expression
> engine pre-installed and pre-built from the C sources.
>
> =A0 To get the Oniguruma C sources and built the engine use these steps:
>
> =A0 1) Download the Oniguruma source package
>
> =A0$ wgethttp://www.geocities.jp/kosako3/oniguruma/archive/onig-5.8.0.tar=
.gz
>
> =A0 2) Decompress and unpack the source package
>
> =A0$ tar zxvf onig-5.8.0.tar.gz
>
> =A0 3) Change current directory to Oniguruma
>
> =A0$ cd onig-5.8.0/
>
> =A0 4) Run configure
>
> =A0$ ./configure
>
> =A0 5) Build Oniguruma engine from C sources
>
> =A0 $ make
>
> =A0 6) Install Oniguruma
>
> =A0 $ sudo make install
>
> =A0 =A0That's it. Now retry to install the gem.
>
> =A0$ gem install oniguruma =A0 or
> =A0$ gem install slideshow
>
> =A0 =A0Cheers.
>
> [1]http://oniguruma.rubyforge.org
> [2]http://ultraviolet.rubyforge.org
>
