PS: Note, that I have used

@hash[ key ].nil? ? @hash[ key ]: default_value  (or better?
@hash.fetch( key, default_value )

on purpose. If you change it to the popular Ruby-esque idiom

@hash[ key ] ||= default_value

you will break the code because false which is a valid value will get
overwritten by the default_value (e.g. if you set lets say
code_line_numbers to false it will get overwritten with the default
value true). Otherwise, thanks for cleaning up the code using more
Ruby idioms.
