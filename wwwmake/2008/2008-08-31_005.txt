Hello,

While creating my presentation, I often like to generate it and see 
how it looks (with the background theme, and my CSS styles, etc.)

You know, the typical "what you see is what you get" mentality.

However, each time I reload the HTML slides after regenerating them, 
the S6 library takes me back to the first slide!  So now I'm forced 
to hit the "page down" key 8 times (or use the jump menu) to get 
back to the slide I was previously looking at.  Not fun.

So I put together a small javascript hack that will take me directly 
to any slide number that is specified in the URL of the browser. 
Just add this code to the bottom of your presentation source, and 
put the slide number (preceded by a '#' symbol) you want to see in 
your browser URL.

Cheers.

------8<-------

<script type="text/javascript">
// access any slide by specifying its number in the URL, like this:
//
//   file://your/slides.html#NNN  (where NNN is the slide number)
//
$(document).ready(function() {
   var num = parseInt( window.location.hash.substring(1) );

   if (num != NaN)
   {
       goTo( num );
   }
});
</script>
